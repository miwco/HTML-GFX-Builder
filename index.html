How it works

Choose, tweak, go on air.

A guided wizard builds the whole graphic from choices — no code required. Professionals can open the always-available editor and take full control of the same file.

01 — CREATE

Start from 43 broadcast-grade templates

Lower thirds, tickers, scoreboards, countdowns, credits, quiz graphics and more — in consistent style families, so your package reads as one show. Or describe any graphic and let AI write it.

02 — MAKE IT YOURS

Your brand, without opening code

Custom colors, imported fonts (embedded in the export), your logo, position, size, and animation — all from live panels. Every change writes real, readable code you can inspect and edit.

03 — RUN ANYWHERE

Export a working package

One click gives you a plug-and-play package: a browser-source overlay for OBS and vMix, an SPX template, a CasparCG single file, or an OGraf graphic — validated, offline-ready, fonts and images bundled.

The whole pipeline

One graphic, start to on-air.

Panels, code, timeline, logic, and export all edit the same file — nothing hides behind a scene model. Scroll through what that feels like.

01 — DESIGN VISUALLY

Style it from live panels

Colors, fonts, position, scale — picked in panels, written as real CSS you can read. Swap the accent and the whole graphic follows.

02 — EDIT THE CODE

Open the code any time

Every graphic is clean, commented HTML/CSS/JS. Change a line and the preview reacts — the panels never hide anything from you.

03 — ANIMATE ON THE TIMELINE

Direct the motion

Entrances, holds, and exits on a real timeline — broadcast-tuned presets, editable down to the tween.

04 — ADD LOGIC

Make it operable

Multi-step graphics, auto-generated operator panels, live data from Google Sheets — a Continue press reveals the next step on air.

05 — EXPORT ANYWHERE

Run it in your studio

One validated, plug-and-play package per target: OBS, vMix, SPX Graphics, CasparCG, OGraf — fonts and images bundled, offline-ready.
